이력서

안녕하십니까 저는 웹 개발에 관심이 많은 대학교4학년 학생입니다. 주 분야는 웹 퍼블리싱 및 웹 프론트입니다.
대학교에서 기본적인 씨언어, C++, 자바 등을 배웠습니다. 그리고 웹 클라이언트단 언어는 Html, Css, JavaScript, Jquery를 공부하고 서버단 언어인 Php를 사용해봤습니다. 데이터베이스는 Mysql과 Firebase를 사용해봤습니다. Css 프레임워크는 Bootstrap과 Vuetify.js를 사용해봤습니다. 웹페이지의 전반적인 퍼블리싱이 가능하고 데이터베이스 연동 경험이 있습니다. 현재는 JavaScript 프레임워크인 Vue.js를 공부하고 있습니다. 버전관리를 위해 git과 gitgub을 사용하고 있습니다.
